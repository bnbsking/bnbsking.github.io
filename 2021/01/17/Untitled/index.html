Markdown tutorial

Markdown is a language for record that widely used in many areas such as writing articles on websites, “README.md” in github, handouts produced by Jupiter notebook, and so on. You can easily write your first article by Markdown after learning it in half an hour. Try to type the text with following styles by the hints on the free online markdown editor. e.g.
https://markdown-editor.github.io/

The text between one star is Italic

The text between two star is bold

The text between three star is Italic and bold

The text between two tilde is strikethrough

The text between tag u is the underline

font tag with color=#FF0000 attribute

font tag with color=#00FF00 attribute

font tag with color=#0000FF attribute
