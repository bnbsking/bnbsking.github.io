  • disadvantage

    • cannot login remotely
    • does not have backend
    • GitHub Pages sites have a soft bandwidth limit of 100GB per month.
    • GitHub Pages sites have a soft limit of 10 builds per hour.
    • static

  • setup process (windows10)

    • register for github

    • click the button “start a project”

    • create the repository with the name [.github.io]

      • select a theme

    • install node.js

    • install git

    • install notepad++

    • npm install -g hexo-cli

    • hexo init name

    • cd name //進入您剛剛建立的 name 資料夾當中，cd 是 change directory 的意思

    • npm install //安裝 Hexo

    • npm install hexo-deployer-git –save //安裝 git 部署套件

    • _config.yml
      hexo cl //清除之前建立的靜態檔案，也可以輸入 hexo clean
      hexo g //建立靜態檔案，也可以輸入 hexo generate
      hexo d
